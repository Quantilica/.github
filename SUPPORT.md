# Suporte — Quantilica

## Onde buscar ajuda

### Bugs e comportamento incorreto
Abra uma **Issue** no repositório do pacote afetado (ex: `github.com/Quantilica/sidra-fetcher/issues`).

Use o template **Bug Report** e inclua:
- Versão do Python e do pacote
- Comando ou código que reproduz o problema
- Mensagem de erro completa (log do `quantilica-core` quando disponível)

### Dúvidas de uso e discussões gerais
Use o **GitHub Discussions** do repositório relevante. Para dúvidas que atravessam múltiplos pacotes, abra em [`Quantilica/.github/discussions`](https://github.com/orgs/Quantilica/discussions).

### Novas fontes de dados
Abra uma **Issue** usando o template **Feature Request / Nova Fonte** no repositório mais próximo do domínio (ex: `sidra-fetcher` para dados IBGE).

## Expectativas de resposta

Este é um projeto mantido em tempo livre. As respostas são feitas em regime de **melhor esforço**, tipicamente em 1-2 semanas. Issues com reprodução clara e completa têm prioridade.

## O que não reportar aqui

- Vulnerabilidades de segurança — veja [SECURITY.md](SECURITY.md)
- Contribuições de código — veja [CONTRIBUTING.md](CONTRIBUTING.md)
